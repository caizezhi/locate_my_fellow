# locate_my_fellow
A demo project for learning.

**准备考研中,项目无限期停止更新**
**本项目仅为个人学习项目,代码质量较差,不具有参考价值**

核心包括 Django, Baidu LBS API 以及个人开发的 [hfut_stu_lib](https://github.com/evilerliang/hfut-stu-lib)

根据个人教务信息定位并将所有人显示在地图上, 主要功能包括数据的统计, 展示与分析并且提供了简单的隐私权限开关.
